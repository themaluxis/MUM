Service Setup

Select and configure media service plugins to connect to your servers. Each plugin supports different features and authentication methods.

Service Information
Plex Media Server: Full-featured media server with user management, sharing, transcoding, and mobile apps. Recommended for most users.
Emby/Jellyfin: Open-source media servers with similar features to Plex. Good alternatives with more customization options.
Kavita: Specialized server for comics, manga, and digital books. Perfect for comic book collections.
AudiobookShelf: Dedicated audiobook and podcast server with chapter support and progress tracking.
Komga: Comic and manga server with advanced metadata management and reading features.
RomM: Retro gaming ROM management server for organizing and playing classic games.
